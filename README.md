## Getting Started

ClypAI is built with **Next.js** and optimized for a fast, modern DX. 
Follow the steps below to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/hyeprlabs/clypai.git
cd clypai
```

### 2. Install Dependencies

This project uses pnpm as package manager.

```bash
pnpm install
```

### 3. Configure Environment Variables

Create a **.env.local** file in the project root and add the required environment variables from **.env.example**!

```bash
cp .env.local
```

### 4. Start the Development Server

```bash
pnpm dev
```

### 5. Open the App

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Your local ClypAI development environment is now running.
Any changes you make will hot-reload automatically.
